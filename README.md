вңҰ Overview

Vireon Imperial Archive is a custom-built personal hub inspired by ancient Chinese imperial aesthetics.

Instead of a traditional link page, this project transforms personal information into an interactive archive containing:

- Social connections
- GitHub achievements
- AniList anime & manga statistics
- Last.fm music activity
- Personal projects
- Custom imperial visuals

Built with a lightweight frontend architecture designed for GitHub Pages deployment.
